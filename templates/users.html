Si el admin quedó bloqueado, recupéralo desde Railway sin tocar la base de datos:

  1. En Railway Dashboard → tu proyecto → Variables, agrega:
PROJECT_TRACKER_ADMIN_PASSWORD=tu-nueva-contraseña

  1. Haz un Redeploy (o espera el próximo deploy).
  2. Inicia sesión con el usuario admin y la contraseña nueva.
  3. Importante: elimina la variable PROJECT_TRACKER_ADMIN_RESET después, o la contraseña se sobreescribirá en cada deploy.
